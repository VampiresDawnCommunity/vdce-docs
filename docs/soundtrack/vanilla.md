
# Soundtrack

## Allgemeine Infos

Diese Seite ist eine Auflistung aller in der Originalversion von *Vampires Dawn - Reign of Blood* verwendeten Musikstücke (MIDIs) und deren Verwendung im Spiel. Sofern bekannt, werden die ursprünglichen Quellen, von denen die MIDI-Dateien sequenziert bzw. konvertiert wurden, mitaufgelistet.

**Hinweis:** Diese Liste enthält ausschließlich Stücke, die auch im Spiel eingesetzt wurden. Im Spielordner des Original-Spiels befindet sich eine Vielzahl weiterer Dateien, die regelmäßig zum *"Soundtrack"* von *Vampires Dawn* gezählt werden, aber im ersten Teil keine Verwendung fanden.

Eine aktuelle Version dieses Dokuments (mit zusätzlichen Verweisen) ist auf *Makerpendium.de* abrufbar:

- [Soundtrack von Vampires Dawn auf *makerpendium.de*](https://www.makerpendium.de/index.php/Soundtrack_von_Vampires_Dawn_-_Reign_of_Blood)

Weitere Infos über den Soundtrack (auch nicht verwendete Tracks & Inhalte aus **VD2**) wurden von *Xenoras* dokumentiert:

- [YouTube-Video von *Xenoras* über den Vampires-Dawn-Soundtrack](https://www.youtube.com/watch?v=IornUfaoprI)
- [Link zum zugehörigen Spreadsheet auf *Google Docs*](https://docs.google.com/spreadsheets/d/1OyZDUkK2MIxz5sfOPspwxab00I-4_F52)
